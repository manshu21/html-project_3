# Image Card with Gradient Button

This project is a simple and responsive **image card layout** built using **HTML and CSS**.  
It displays an image with a **full-width gradient button** below it, including a smooth hover effect.

---

## 🚀 Features

- Clean and minimal UI
- Responsive layout using `vh` and `vw`
- Gradient button design
- Smooth hover and active effects
- No Flexbox or Grid used (pure CSS)

---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

## 📂 Project Structure

---

## 📄 HTML Code Overview

- Uses semantic HTML structure
- Image and button wrapped inside a container
- Button width automatically matches the image width

---

## 🎨 CSS Styling Highlights

- `vh` and `vw` units for responsive spacing
- Linear gradient background for the button
- Smooth hover transition with shadow and lift effect
- Box shadow for card depth

---

## 💡 Hover Effect

On hovering the button:

- Background color changes
- Button slightly lifts up
- Shadow becomes more prominent

---

## 📱 Responsive Behavior

- Layout adapts to different screen sizes
- Button height scales using viewport units
- Image remains fully responsive
